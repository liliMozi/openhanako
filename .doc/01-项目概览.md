# OpenHanako 项目概览

## 一句话介绍

OpenHanako 是一个**有记忆、有性格的个人 AI 助理**——它不仅能聊天，还能记住你说过的话、拥有自己的情绪和人格、帮你做事、学习新技能、接入多个平台。

## 核心特性

1. **记忆系统** — 用 SQLite FTS5 全文搜索存储记忆，Agent 能记住跨对话的信息
2. **人格系统** — 三种"源"（yuan）人格模板：Hanako（意识流四池）、Butter（体感三拍）、Ming（沉思两层）
3. **多 Agent** — 支持多个 AI 角色，各有独立记忆和性格，可以互相聊天
4. **Skill 系统** — Agent 可以学习新技能，技能用 Markdown 描述，支持从 GitHub 安装
5. **工具调用** — 搜索、浏览网页、定时任务、文件操作、浏览器控制等 15+ 工具
6. **书桌系统** — Agent 有自己的工作空间，可以定时巡检文件、执行 cron 任务
7. **多平台接入** — 桌面 + Telegram + 飞书 + QQ
8. **沙盒安全** — macOS seatbelt / Linux bwrap / Windows PathGuard
9. **流式解析** — 实时解析 AI 输出中的 `<mood>`、`<think>`、`<xing>` 等标签

## 目录结构总览

```
OpenClaw-OpenHanako/
├── core/           # 核心引擎层 — Agent、Session、模型、配置、技能管理
├── lib/            # 基础库层 — 记忆、沙盒、工具、Bridge 适配器、LLM 调用
├── hub/            # 消息调度中枢 — 路由、事件总线、定时任务、频道
├── server/         # 后端服务 — Fastify HTTP + WebSocket + REST API
├── desktop/        # 桌面应用 — Electron 主进程 + React 前端
├── skills2set/     # 内置技能 — skill-creator、canvas-design、quiet-musing
├── scripts/        # 构建脚本 — 启动、打包修复、MinGit 下载
├── tests/          # 测试 — Vitest 单元测试
├── devlog/         # 开发日志
├── index.js        # 根入口
├── package.json    # 依赖和脚本
└── vite.config.ts  # Vite 构建配置
```

## 运行命令

| 命令 | 说明 |
|------|------|
| `npm install` | 安装依赖（会自动 electron-rebuild） |
| `npm start` | 构建前端并启动 Electron |
| `npm run start:dev` | 开发模式启动 Electron |
| `npm run start:vite` | Vite HMR 模式启动（推荐开发用） |
| `npm run dev:renderer` | 仅启动 Vite 开发服务器 |
| `npm run server` | 启动独立服务端（附带 CLI） |
| `npm run cli` | 启动命令行界面 |
| `npm test` | 运行测试 |
| `npm run dist` | 构建 macOS DMG |
| `npm run dist:win` | 构建 Windows NSIS 安装包 |

## 数据目录

运行时数据存储在 `~/.hanako/`（开发模式为 `~/.hanako-dev/`）：

```
~/.hanako/
├── agents/              # 每个 Agent 的独立数据
│   └── {agent-id}/
│       ├── config.yaml      # Agent 配置
│       ├── identity.md      # 身份设定
│       ├── ishiki.md        # 意识/行为准则
│       ├── pinned.md        # 置顶记忆
│       ├── memory/          # 记忆数据
│       │   ├── facts.db         # SQLite FTS5 元事实数据库
│       │   ├── memory.md        # 编译后的记忆（注入 system prompt）
│       │   ├── today.md         # 今天的记忆
│       │   ├── week.md          # 本周的记忆
│       │   ├── longterm.md      # 长期记忆
│       │   ├── facts.md         # 重要事实
│       │   └── summaries/       # 每个 session 的滚动摘要
│       ├── sessions/        # 对话记录
│       ├── desk/            # 书桌系统（cron、活动）
│       ├── experience/      # 经验库
│       ├── learned-skills/  # 自学技能
│       └── avatars/         # 头像
├── user/                # 用户数据（跨 Agent 共享）
│   └── user.md              # 用户自我描述
├── channels/            # 频道数据
├── skills/              # 全局技能
├── auth.json            # API 认证
├── models.json          # 模型注册
├── providers.yaml       # Provider 凭证
└── preferences.json     # 全局偏好
```
